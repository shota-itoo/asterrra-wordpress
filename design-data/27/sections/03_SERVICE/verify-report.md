## 修正履歴

### Round 1 — グリッドが2列にならず4枚縦積み（症状: グリッド崩れ）

- **原因**: Inner コンテナ（max-width 1152 / box-sizing:border-box）の左右 padding 16px が 1152 の content box を 1152−32=**1120px** に縮め、566×2+gap20=1152px のカード行が収まらず flex-wrap で1枚ずつ折り返していた。grid 実測幅 1120px・カード4枚すべて x=437 で縦積み（セクション高 2959px）。
- **対処**: `tree.js` の Inner の `spacingPadding` 左右を 16 → **0** に変更（datapack の「コンテンツ幅1152・左右paddingは安全余白のみ」の意図に沿い、1152 を満額グリッドへ渡す）。上下 padding 96px は維持。テキスト・属性以外は不変。
- **結果**: grid 幅 1152px・2×2 グリッド成立（セクション高 1659px に収束）。isValid 維持。

修正ループ 1/3 で収束。
